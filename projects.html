QuBetterParking

QuBetterParking is a simulator designed for the Quinnipiac University Campus parking since there are multiple issues surrounding it. In this simulator each student/faculty member gets assigned a parking ticket which tells them where exactly they should be parking. As each person enters the parking lot an updated view of available spots is shown. When the lot is full the simulator informs the user that the lot is full. Finally, when students/faculty members are exiting the lot, they return their tickets showing that spots are becoming available.

Collaborators: Christopher Anzilotti, MacArthur Mills
